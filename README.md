Responsi PBO - E-Commerce Cart
Aplikasi dikembangkan dengan:
- Penyimpanan data menggunakan MySQL
- Implementasi Event Diskon 12.12
- Penerapan MVP Pattern
- Repository Pattern
- Strategy PatterN

Pattern yang Digunakan
 MVP (Model View Presenter)
- View → CartView
- Presenter → Responsi.java
- Model → Repository & DTO
Repository Pattern
Digunakan untuk memisahkan akses database dari logic aplikasi
Strategy Pattern
Digunakan untuk sistem diskon agar mudah dikembangkan
